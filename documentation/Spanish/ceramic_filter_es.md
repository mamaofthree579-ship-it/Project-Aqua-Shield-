# Filtro Cerámico de Aserrín + Arcilla
Alternativa cuando no existen filtros cerámicos comerciales.

## Materiales
- Arcilla  
- Aserrín fino o cáscara de arroz  
- Molde simple  
- Horno, barril o fogón para cocer

## Pasos
1. Mezclar 1 parte de aserrín + 3 partes de arcilla.  
2. Agregar agua hasta formar masa firme.  
3. Moldear un tazón o cono.  
4. Secar 2–3 días.  
5. Cocer a 850–900°C (el aserrín se quema → deja poros).  
6. Opcional: recubrir con plata coloidal.

## Qué elimina
✔ Sedimentos  
✔ Muchas bacterias  
✔ Algunos parásitos

## Qué NO elimina
✘ Metales  
✘ Químicos  
✘ Salinidad
