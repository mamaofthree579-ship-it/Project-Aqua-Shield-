# Filtro Familiar de Arena + Carbón (Cubetas)
Rápido, escalable y útil para familias o comunidades.

## Materiales
- 2 cubetas o bidones
- Arena limpia
- Grava gruesa y fina
- Carbón vegetal triturado
- Tela o malla
- Llave o agujero para salida

## Capas (de arriba hacia abajo)
1. Tela  
2. Grava gruesa  
3. Grava fina  
4. Carbón (5–8 cm)  
5. Arena (15–25 cm)  
6. Tela

## Armado
1. Hacer un agujero para goteo.  
2. Colocar capas en orden.  
3. Verter agua lentamente.  
4. Recoger en la segunda cubeta.

## Seguridad
Funciona mejor con **desinfección secundaria**.
